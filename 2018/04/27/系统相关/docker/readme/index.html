127.0.0.1与0.0.0.0的区别

127.0.0.1只能本机访问，无法通过它对外提供服务。

0.0.0.0 则表示本机所有ip。如本机有192.168.0.2，10.0.0.2和127.0.0.1三个ip,使用它表示同时监听这些ip的某个端口。
